# JavaScript Classes Index

This folder contains a list of all JavaScript classes found in the CRMBA project's JS source folders. Each file in this folder documents the name and location of a class, serving as a quick index for developers and tools.

- Each class is listed with its file path.
- Use this as a reference to locate class definitions quickly.

> To add a new class, create a markdown file named after the class and list its location(s).
